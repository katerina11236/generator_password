 #Менеджер паролей (самохост): Генерация, шифрование; бэк — AES, 2FA.

		Введение:

Разрабатываемое программное обеспечение представляет собой саморазмещаемый (self-hosted) менеджер паролей, предназначенный для безопасного хранения, генерации и управления учетными
данными как индивидуальным пользователям, так и организациями. В условиях роста числа киберугроз и участившихся случаев утечек данных из облачных хранилищ, использование локально 
развертываемого решения обеспечивает необходимый уровень контроля и защиты конфиденциальной информации.
 
		Обзор:

		Цель спецификации.

Обеспечение конфиденциальности и целостности хранимых учетных данных пользователя путем применения криптографических методов защиты информации,
благодаря основному международному симметричному алгоритму блочного шифрования - AES.
Данный метод широко используется по всему миру: от смартфонов до военных систем. При возможности перехвата файла с базой данных паролей с сервера,
злоумышленник не сможет прочитать ни одного пароля без мастер-пароля.

Применение метода двухфакторной аутентификации(2FA) для усиления защиты процесса аутентификации и предотвращения несанкционированного доступа 
к расшифрованным данным (даже при компрометации мастер-пароля).
Двухфакторная аутентификация требует подтверждения личности с использованием двух независимых факторов, что существенно снижает риски 
несанкционированного доступа.

Разработка API-приложения, для хранения и генерации паролей. Реализация в виде API-приложения позволяет централизовать логику безопасности на 
серверной стороне, обеспечивая единые стандарты шифрования и аутентификации для всех клиентских приложений.

		Объём проекта:

		Краткое описание проекта.

Необходимо разработать программу для хранения и генерации паролей, которую пользователь сможет установить на свой собственный сервер. 
Программа будет хранить все пароли в зашифрованном виде, чтобы никто посторонний не мог их прочитать, а также помогать пользователю создавать
сложные и надежные пароли. Для входа в программу потребуется не только пароль, но и дополнительное подтверждение через специальное приложение на 
телефоне (двухфакторная аутентификация). Также программа будет предоставлять программный интерфейс (API), чтобы другие программы могли с ней 
взаимодействовать.

		#Общее описание

Программа предназначена для людей и компаний, которые хотят хранить свои пароли в безопасности, а также получать помощь в создании новых 
надёжных паролей. При этом пользователи полностью контролируют место хранения своих данных.

		Целевая аудитория:

1. Физические лица - обычные люди, у которых много аккаунтов в интернете. Им нужно место, где можно хранить все пароли и заходить с телефона или компьютера.
	
	Как обычный пользователь я хочу сохранять свои пароли в одном месте,  
	1. На главном экране я нажимаю кнопку "Добавить"  
	2. Автоматически перехожу в диалоговое окно  
	3. Я выбираю сайт для которого сохраняю пароль, ввожу логин и пароль, также по желанию могу добавить его в личную папку для быстрого навигирования на область задач.  
	4. После заполнения полей, я добавляю пароль путём нажатия кнопки в диалоговом окне "Добавить пароль".  
	5. При переходе на главную страницу приложения я могу увидеть свой сохранённый пароль.  
   (Также я могу автоматически сгенерировать пароль. Для этого
	1. На главной странице нажимаю на кнопку "Создать"  
	2. Автоматически перехожу в диалоговое окно  
	3. Задаю начальные параметры для него (пароль может состоять из: цифр, прописных букв, строчных букв, специальных символов), в том числе его длину.  
	4. Нажимаю на кнопку "Добавить пароль" внизу диалогового окна  
	5. Пароль я могу посмотреть на главной странице.)*  
Теперь я могу не держать пароли в голове и не записывать в блокнот.

2. Семьи - несколько человек, которым нужен общий доступ к домашним паролям (Wi-Fi, подписки на кино, умный дом). Каждый входит под своим логином, но видит общие пароли.

Как член семьи	я хочу видеть общие пароли (Wi-Fi, подписки),  
	1. Перехожу по временной ссылке к каталогу с общими паролями (с разрешённым количеством переходов по ней)  
	2. Добавляюсь в пользовательскую группу  
Теперь я могу пользоваться общими паролями из этого каталога, также я могу пользоваться только одним предоставленным паролем по ссылке. Все пароли будут выводиться на главной странице, в разделе каталоги.
Я не могу менять эти пароли и повторно делиться(в рамках системы). Таким образом я могу не просить каждый раз пароли у других.

3. Технически подкованные пользователи - люди, у которых дома стоит свой сервер. Они не хотят хранить данные в интернете и предпочитают всё контролировать сами. 
	Как пользователь со своим сервером я хочу настроить регулярное резервное копирование,  
	1. Нажимаю на главной странице кнопку "Настройки"  
	2. Автоматически перехожу в диалоговое окно  
	3. Нажимаю кнопку "Интервал автосохранения паролей"  
	4. Автоматически перехожу в диалоговое окно с выбором количества сохранении паролей в месяц  
(также я могу поменять папку для сохранения паролей на ПК)  
	5. Нажимаю на копку "Сохранение" для завершения настройки.  
   
Теперь у меня настроено копирование паролей чтобы их не потерять при сбое.

		Требования к функционалу:
Приложение должно обеспечить пользователю следующие функции  
	
добавление: Данная функция осуществляется посредством нажатия на кнопку "Добавить" на главной странице. В автоматически открывшемся диалоговом окне пользователь вводит название сайта, логин и пароль. Программа сохраняет это в зашифрованном виде.  

удаления пароля: Данная функция осуществляется нажатием на необходимый пароль для удаления на главной странице. В автоматически открывшемся диалоговом окне, есть кнопки взаимодействия с записью, после нажатия кнопки "Удалить пароль", пароль и все связанные с ним данные удаляются. Таким образом можно удалить ненужную запись.  

просмотр паролей: Данная функция позволяет пользователю видеть список всех сохранённых записей. Это можно осуществить, находясь на главной странице.        

редактирование: Данная функция осуществляется нажатием на необходимый пароль для редактирования на главной странице. В автоматически открывшемся диалоговом окне, есть кнопки взаимодействия с записью, после нажатия кнопки "Редактировать", можно изменить выбранный пароль. Таким образом можно изменить любой сохранённый пароль или его описание  

поиск: Данная функция осуществляется путём нажатия на кнопку "Найти" на главной странице.В автоматически открывшемся диалоговом окне можно ввести в поисковую строку название сайта или его часть, таким образом можно быстро найти нужный пароль.  

категории: Данная функция осуществляется путём нажатия на кнопку "Каталоги" на главной странице. В автоматически открывшемся диалоговом окне можно добавить новую категорию или удалить, также есть функция редактирования названии каталогов.  

импорт и экспорт: Данные функции осуществляются нажатем на кнопки "Импорт"/"Экспорт" на главной странице. После выбора существующего файла с паролями они отобразятся на главной странице, при импорте паролей, файл будет сохраняться в директорию ПК, которая указана в настройках.Таким образом можно перенести пароли из другой программы или сохранить их в файл (например, для резервной копии)  

создание случайного пароля: Данная функция осуществляется путем нажатия на кнопку "Создать", которая находится на главной странице.В автоматически открывшемся диалоговом окне можно выбрать праметры для генерации пароля, после чего следует добавить запись, путём нажатия кнопки "Добавить пароль". Программа сама придумывает пароль, который трудно взломать.  

проверка надёжности: Данная функция будет реализована после нажатия кнопки "Добавить", и указания всей необходимой информации о пароле. В поле "Оценка пароля" отображается насколько сложный код придумал пользователь (слабый, средний, надёжный) для рекомендации по улучшению уровня надёжности.  

копирование в буфер: Сгенерированный пароль можно сразу скопировать, чтобы вставить на сайте. Это достаточно сделать путём щёлка левой кнопки мыши по паролю.  

вход в программу: Пользователь вводит мастер-пароль при запуске программы.  

двухфакторная аутентификация: После ввода пароля программа просит подтверждения через почту для совершения входа.  

настройка 2FA: На главной странице после нажатия кнопки "Настройки". В автоматисески открывшемся диалоговом окне можно настроить 2FA путём нажатия на кнопку "Влючить/Выключить 2FA". Таким образом пользователь может включить или отключить двухфакторную аутентификацию у себя в настройках.  

выход из системы: Программа закрывает доступ к паролям, если пользователь вышел из системы  

общий доступ: можно дать другому пользователю доступ к определённой папке или паролю.  

права доступа: можно разрешить только смотреть пароль или разрешить редактировать.
